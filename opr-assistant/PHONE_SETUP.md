# OPR Play Assistant

This folder is a first working web-app version of the OPR helper.

## Test On This Computer

Open `index.html` in a browser, or double-click `Start OPR Assistant.bat` and use the link it prints.

## Test On iPhone Over Wi-Fi

1. Keep your computer and iPhone on the same Wi-Fi.
2. Double-click `Start OPR Assistant.bat`.
3. Open the computer's Wi-Fi address on the iPhone.
4. In Safari, tap Share.
5. Tap Add to Home Screen.

## Long-Term Phone Use

For the smoothest phone version, host this folder with GitHub Pages, Netlify, Vercel, Cloudflare Pages, or Codex Sites, then open that link on iPhone Safari and add it to the home screen.

## Current Features

- Add units manually.
- Import pasted Army Forge text or simple JSON.
- Track models, wounds, activation, ready/shaken/stunned status.
- Calculate hit and defense target numbers.
- Apply AP, failed saves, Regeneration, and Tough(X).
- Undo recent changes.
- Save state locally in the browser.

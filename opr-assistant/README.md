# OPR Play Assistant

A phone-friendly One Page Rules play assistant for imported Army Forge text rosters.

The app runs entirely in the browser and stores game state on the device.

## GitHub Pages

If these files are uploaded into an `opr-assistant` folder in the `hillerbillee/hillerbillee-projects` repository, the Pages URL will usually be:

`https://hillerbillee.github.io/hillerbillee-projects/opr-assistant/`

## Phone Setup

1. Open the GitHub Pages URL on your iPhone.
2. Tap Share.
3. Tap Add to Home Screen.

## Import Format

Use the Army Forge copied text format:

```text
Unit Name [5] Q4+ D5+ | 100pts | Regeneration, Tough(3)
5x Weapon Name (12", A2, AP(1))
```
